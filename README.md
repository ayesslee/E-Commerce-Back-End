# E-Commerce Back-End

## Description

Backend application for an e-commerce site using mysql.


## Installation

![gif of app](/images/sql.gif)

`npm i`

`mysql -u root -p`

Enter password when prompted

`source db/schema.sql`

`quit`

`npm run seed`

`npm start`
